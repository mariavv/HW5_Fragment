# HW5_Fragment

1. Создать проект
2. Используя компонент ViewPager в сочетании с FragmentPagerAdapter создать экран, где горизонтальным скроллом можно будет выбрать один из трех подэкранов.
3. Первый подэкран должен показывать текущее время, второй - текущую дату и третий - произвольное изображение.
4. Данные во фрагментах должны быть актуальны на момент открытия пользователем (перелистывания на соответствующий фрагмент)
5. Пример макетов экранов находится в презентации
